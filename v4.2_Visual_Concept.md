### 2. Domain Badge in Header

After selecting Customer Care, the header shows:

```

```

## How It Changes by Role

### Builder + Customer Care

- Checklist: Agent creation steps
- Templates: Ticket routing, sentiment analysis
- Recommendations: Zendesk integration

### Admin + Procurement

- Checklist: Tenant setup steps
- Templates: PO approval, vendor onboarding
- Recommendations: SAP integration

### SME + HR

- Checklist: Workflow creation steps
- Templates: Employee onboarding, leave management
- Recommendations: Workday integration

---

## Key Visual Elements

### Domain Badge (Purple)

- Background: #8a3ffc
- Shows domain icon + name
- Always visible in header

### Smart Recommendations (Purple Panel)

- Background: #f0e6ff
- Border-left: 3px solid #8a3ffc
- Shows detected integrations
- Shows popular choices

### Domain Tags

- On checklist items
- Shows which domain you're in
- Purple background (#f0e6ff)
