What We Learned

From 5 User Interviews (240 Minutes)

We spoke with Client Engineering, Account Technical Leaders, Customer Success, Sales Leadership, and Ecosystem Architects. The message was unanimous:

"Personalization is essential. Templates must be tuned to the user's specific interests and use cases."
— Validated by all 5 interviews

Key Insights

  • Integration is the #1 blocker - Users get stuck on ServiceNow, Salesforce, SAP, Workday connections
  • Empty analytics provide no value - Day 0 users have no agents, so analytics are meaningless
  • Chat interface needs clarity - Users don't immediately recognize conversational capabilities
  • Experienced users need issue-first dashboards - Sales leaders want to see problems, not general metrics
  • Time to first agent is everything - Users have already seen the product; they want to DO something

From Stakeholder Consensus

Ronak (PM) - "Activation Over Education"

"First time users would have already seen the product. So, the goal shouldn't be helping them learn about the product but quickly get them started to an agent."

Frances (Manager) - Hybrid Approach

"Blend of direction 1 and 2, where we ask specifying questions and use the chat to curate the user's experience."

Consensus: Hybrid approach combining AI-first conversational experience with role-based structure.

Competitive Intelligence

Analysis of 7 leading SaaS products (Intercom, Hubspot, Servicenow, Salesforce, Notion, Atlassian, Clickup) reveals 5 critical patterns:

1. Progress Visibility

Intercom: "0/5 steps" - Users know exactly where they are and what's left

2. Progressive Disclosure

Intercom: Expand current step, collapse others - Dramatically reduces cognitive load

3. Multi-Purpose Chat

Intercom: Single panel with 4 tabs (Messages, Help, News, Support) - Consolidates all communication

4. Domain Personalization

Hubspot: Chart-free, domain-personalized insights - Industry-specific templates and content

5. Contextual Help

Salesforce: Agent for contextual help - Help user where they are

Opportunity: Best of Both Worlds

Intercom's UX patterns + watsonx Orchestrate's personalization depth = Best-in-class onboarding
